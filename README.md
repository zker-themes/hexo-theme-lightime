## 特色

- 单栏，简洁。
- 集成Gitment和Disqus评论系统。使用InstantClick实现无刷新浏览，更快更流畅。
- 评论使用点击加载方式，减少不必要脚本，同时兼容InstantClick。
- 支持Service Worker，但和InstantClick冲突，按需选择。
- 代码高亮。
- 支持归档页，分类、标签页。
- i18n国际化。
